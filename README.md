# 소개

로그인, 회원가입, 게시글 등 CRUD 기능을 작업하기 위한 템플릿입니다.

# 사용 스텍

React, Hook, styled-Component

`yarn start`

프로젝트를 localhost:3000 주소로 개발용 모드로 실행합니다.

`yarn build`

build를 실행합니다.
